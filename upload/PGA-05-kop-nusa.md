## PILLAR 2: KEADILAN (JUSTICE) — Keadilan yang Tidak Memihak

### Definisi dan Makna Mendalam

```markdown
┌─────────────────────────────────────────────────────────────────────────────┐
│ PILLAR 2: KEADILAN (JUSTICE)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ "SETIAP SUARA BERHARGA. SETIAP ANGGOTA MEMILIKI NILAI YANG SAMA."          │
│                                                                             │
│ ══════════════════════════════════════════════════════════════════════════ │
│                                                                             │
│ DEFINISI:                                                                  │
│ Keadilan adalah PRINCIPLE bahwa setiap manusia memiliki nilai yang sama   │
│ — tidak peduli latar belakang, status, atau kekuasaan mereka.             │
│                                                                             │
│ Keadilan bukan berarti "semua orang mendapat hal yang sama."              │
│ Keadilan berarti "setiap orang mendapat apa yang mereka HARUSKAN."        │
│                                                                             │
│ Dalam KNBMP, keadilan berarti:                                             │
│ • 1 anggota = 1 suara (bukan 1 modal = 1 suara)                           │
│ • Akses yang sama untuk semua anggota                                      │
│ • Distribusi SHU yang fair berdasarkan kontribusi                         │
│ • Kesempatan yang sama untuk berkembang                                   │
│ • Perlindungan yang sama dari eksploitasi                                 │
│                                                                             │
│ ══════════════════════════════════════════════════════════════════════════ │
│                                                                             │
│ DIMENSI KEADILAN:                                                          │
│                                                                             │
│ 1. KEADILAN DISTRIBUTIF                                                    │
│    Sumber daya dan hasil dibagikan berdasarkan:                            │
│    • Kontribusi (bukan hanya modal)                                        │
│    • Kebutuhan (untuk yang rentan)                                        │
│    • Merit (untuk yang berprestasi)                                        │
│                                                                             │
│ 2. KEADILAN PROSEDURAL                                                     │
│    Proses pengambilan keputusan berlangsung adil:                          │
│    • Semua suara didengar                                                 │
│    • Aturan diterapkan konsisten                                          │
│    • Tidak ada favoritisme                                                │
│                                                                             │
│ 3. KEADILAN INTERAKSIONAL                                                  │
│    Interaksi antar-anggota berlangsung dengan hormat:                     │
│    • Tidak ada intimidasi                                                 │
```
